# Reach Them AI - Review Management App

A Next.js application for managing online reviews and reputation with AI-powered review advisor.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Review removal assistance
- AI-powered review advisor
- Multi-platform support (Google, Yelp, Facebook, Trustpilot)
- Stripe payment integration
- Supabase authentication

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Supabase
- Stripe
