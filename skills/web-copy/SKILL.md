# Web Copy Local Seo

## Overview

This skill provides specialized guidance for crafting high-converting landing page copy and recommending matching web design elements for a local SEO/digital marketing agency. It ensures consistency with the company's branding: dark blue gradients (#001F3F to #0074D9), light blue accents (#7FDBFF), white text, clean sans-serif fonts, modern hero sections with bold headlines.

## Instructions

When generating landing page content:

1. **Structure**: Follow AIDA (Attention, Interest, Desire, Action) or PAS (Problem, Agitate, Solution). Start with powerful hero headline/subheadline, benefits, social proof, services, CTA.

2. **Copy Best Practices**:
   - Headlines: Benefit-oriented, include keywords like "Local SEO", "Google My Business Optimization", "Local Digital Marketing".
   - Body: Conversational, scannable with bullet points, short paragraphs (<4 lines).
   - CTAs: Action verbs, urgency (e.g., "Get Your Free Audit Today").
   - SEO: Naturally integrate primary/secondary keywords, meta descriptions.
   - Tone: Professional yet approachable, trustworthy for local businesses.
   - Length: Hero 50-100 words, sections 150-300 words.

3. **Design & Aesthetics**:
   - Layout: Hero banner full-width with gradient background, navigation bar, sections with alternating backgrounds, testimonials carousel, FAQ accordion, footer.
   - Colors: Dark blue gradient background (#0A2540 to #0074D9), accents in cyan/blue (#00B4D8), white/ off-white text. Buttons: solid blue with hover effects.
   - Typography: Bold sans-serif for headings (e.g., Inter or similar), readable body font.
   - Visuals: High-quality local business stock images, icons for services, subtle animations.
   - Mobile-first responsive design.

4. **Output Format**:
   - Provide full HTML structure if requested, or section-by-section copy.
   - Suggest image prompts for visuals matching style.
   - Include calls-to-action linked to contact/forms.

Reference the Reach Them AI example for visual consistency: prominent hero with gradient blue, large white text headlines, blue buttons.

Always ask for specifics: target service (e.g., GMB optimization, review management), location, target audience.
